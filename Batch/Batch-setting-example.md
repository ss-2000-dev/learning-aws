# AWS Batch の設定手順

#### ジョブとオーケストレーション設定
以下のうちのいずれか
- Fargate
- EC2
- EKS

#### コンピューティング環境を作成
- コンピューティング環境設定
- インスタンス設定（EC2の場合）
- ネットワーク設定（VPC, Subnet, Security Group）

#### ジョブキューの作成
- 名前と優先度（優先度は1にした）

#### ジョブ定義を作成
- 実行タイムアウト
- コンテナ設定（EC2の場合でも）
- コマンド（オプション）
- 実行ロール（オプション）
- ジョブロール設定（オプション）
- パラメータ
- 環境設定（vCPU, メモリ、GPU）
- Linux, ファイルシステム設定

#### ジョブを作成
（ジョブはECS クラスター内の Amazon ECS コンテナインスタンスで実行されるコンテナ化されたアプリケーションとして実行できる）

### 参考
- [ウィザードを使用した Amazon EC2 オーケストレーションの開始](https://docs.aws.amazon.com/batch/latest/userguide/getting-started-ec2.html)
